# 文献应用映射报告

本文档逐一列出 JobRec_KG 每个模块所应用的参考文献及其具体映射关系。

---

## 1. LightGCN 召回模块 (`src/recall/lightgcn.py`)

| 文献 | 应用的具体实现 |
|------|---------------|
| **LightGCN (R1)** | 3 层图传播 `n_layers=3`；去除特征变换 W 和非线性 σ；Embedding 层叠加取平均 `alpha=1/(K+1)`；64 维嵌入；Xavier 初始化 |
| **BPR (R2)** | `bpr_loss()`：正样本分 > 负样本分的排序优化；负采样策略 `torch.randint` |
| **NCF (R3)** | 作为 LightGCN 选型对比基线（证明去 W 去 σ 有效） |
| **NGCF (R4)** | LightGCN 前身，含 W+σ → 本项目选 LightGCN 的理由 |
| **HOMOPHILY (R5)** | 解释 User-Job 二部图符合同配性 → LightGCN 有效 |
| **SGL (R7)** | 未来升级方向：自监督对比学习增强（预留） |
| **GTF (R8)** | 未来解决 over-smoothing 的方向 |

| 文献 | 应用的具体实现 |

## 4. GAT 技能加权模块 (`src/models/gat.py`, `src/ranking/gat_weighter.py`)

| 文献 | 应用的具体实现 |
|------|---------------|
| **GAT (R23)** | 4 头 × 2 层多头注意力；LeakyReLU `negative_slope=0.2`；Dropout `0.6`；Message Passing 聚合 |
| **GATv2 (R24)** | 动态注意力机制（本项目 GATv2 实现依据）；证明原 GAT 注意是静态的 |
| **HAN (R25)** | 异构图注意力参考 → 本项目 Job-Skill 异构图建模思路 |
| **GNNExplainer (R26)** | GAT 技能权重的可解释性输出（`get_explainability_report()`） |
| **Multi-Relational Poincaré (R27)** | 多关系图嵌入参考 → 技能先修边权重设计参考 |

**16 维特征构造**: `SkillFeatureBuilder.build_from_kg_data()`：
- [0:4] 岗位频率 — 来自 job_associations 统计
- [4:8] 图中心性 — PageRank (50 轮幂迭代, d=0.85) + DC + 平衡比 + PR×DC
- [8:10] 难度编码 — CATEGORY_DIFFICULTY 启发式映射
- [10:14] 市场趋势 — CATEGORY_TREND 启发式 + PR 交互 + 新兴代理
- [14:16] 薪资影响 — CATEGORY_SALARY_IMPACT 启发式 + PR 交互

**伪标签训练**: `0.6·PageRank + 0.4·岗位频率` → MSE Loss → 200 epochs

## 7. 评估与指标模块 (`src/utils/training.py`)

| 文献 | 应用的具体实现 |
|------|---------------|
| **RecSys 离线评估 (R36)** | Recall@K, NDCG@K, Precision@K, MRR@K, HitRate@K, Coverage@K, AUC 共 7 项指标 |
| **ERR (R38)** | 排序质量评估的补充参考 |

指标计算公式:
- Recall@K = |Rec(u) ∩ Test(u)| / |Test(u)|
- NDCG@K = DCG@K / IDCG@K (log₂ 位置衰减)
- AUC = 随机正 > 负样本概率 (采样 1000 近似)
