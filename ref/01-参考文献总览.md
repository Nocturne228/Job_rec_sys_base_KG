## 前沿技术升级方向

每个方向的完整技术原理和融入路径详见下方。摘要如下：

| 方向 | 核心论文 | 融入方式 | 预期收益 |
|------|---------|----------|---------|
| LLM × Recommendation | P5 (Hou et al. WWW 2023); TALLRec (Bao et al. 2023) | Phase A: LLM 排序第 4 因子; Phase B: Top-50→10 重排序 | NDCG@10 +5pp |
| GNN for RecSys | LightGCL (Wang et al. SIGIR 2023); SGL (Wu et al. KDD 2021) | 现有 LightGCN + 对比 Loss | Recall@10 +4pp |
| Vector Search | HNSW (Malkov & Yashunin IEEE TPAMI 2018) | IndexFlatIP → HNSWIP | 10ms→2ms |
| Agentic RAG | ReAct (Yao et al. ICLR 2023); Reflexion (Shinn et al. NeurIPS 2023) | 硬编码→Tool; 加自验证循环 | 幻觉率 8%→2% |

### LLM × Recommendation（端到端 LLM 推荐）

**当前**: 传统"召回→排序"双阶段。**新范式**: 直接用 LLM 做推荐。

核心论文：P5 (Hou et al. WWW 2023) 将推荐任务统一为文本生成；InstructRec (Zhang et al. 2023) 用指令微调；TALLRec (Bao et al. 2023) 用 LoRA 微调 LLaMA。

融入路径：Phase A — 线性融合增加 LLM_Rank 第 4 因子（最小改动）；Phase B — Top-50 用 LLM 重排序（中等改动）。

### GNN for RecSys（自监督图学习）

LightGCL (Wang et al. SIGIR 2023) 通过 SVD 重构增强结构信息；SGL (Wu et al. KDD 2021) 通过随机节点/边丢弃做对比增强。

融入路径：在 LightGCN 训练加对比 Loss — 改动极小（新增 1 个损失项 + 1 个增强模块）。

### Vector Search（HNSW）

HNSW 多层小世界图索引，当前向量检索事实标准。FAISS 支持 `IndexHNSWFlat`，2 行代码从 IndexFlatIP 升级。

融入路径：Demo 保留 IndexFlatIP（全量检索无性能问题）；生产用 HNSW。

### Agentic RAG（LangGraph 工具调用）

LangGraph 0.2 后支持工具调用（ToolNode）+ ReAct 范式（推理-行动-观察循环）+ Reflexion 自我纠错。

融入路径：硬编码 Neo4j 查询 → LLM 自主选择 Tool；生成后加自验证节点。

## 业界招聘平台技术调研

### 知识图谱

- **LinkedIn**: 知识图谱 vs 实体图谱 — 静态本体 vs 动态语料库。Entity Graph 更侧重于文档中实体的近实时全面表示
- **BOSS 直聘**: KPRN — 知识感知路径循环网络。通过图谱路径（用户→技能→职位→...）推理实现可解释推荐
- **本项目映射**: JobRec_KG 的技能先修知识图谱 → GAT 学习技能重要性 → 与 KPRN 思路相通

### 简历解析

- **前沿方案**: 布局感知的多模态深度学习模型，融合 PDF 元数据与 OCR 视觉特征
- **工业实践**: LinkedIn 通过微调轻量 LLM (0.6B) 达到 GPT-4 相当的解析精度
- **本项目映射**: 简历 NER 管线设计（spaCy + RoBERTa）→ 技能词典兜底 → 双管线策略

### 分布式嵌入与对比学习

- **技术路线**: TF-IDF/BM25 → TransR/Word2Vec → Transformer-based → GNN Embeddings
- **LinkedIn STAR 系统**: LLM + GNN 联合微调，端到端信号集成
- **本项目映射**: SBERT 语义召回 → LightGCN 图嵌入 → Ensemble 融合 = 简化版 STAR

### 两端市场反馈

- **Indeed**: 每分钟处理数以万计的交互信号（点击/保存/忽略/申请 + 雇主查阅/邀请/录用）
- **反馈闭环**: 算法自修正 — 频繁忽略某类职位 → 降权；雇主高频录用某特征候选人 → 自动识别隐性特征
- **本项目映射**: 交互权重分层（view/click/save/apply）+ A/B Testing 闭环验证

### 生成式可解释性

- **Indeed + OpenAI**: 微调 GPT → 自然语言解释推荐理由。申请启动率 +20%，面试转化率 +13%
- **本项目映射**: LangGraph + GraphRAG + Qwen-2.5 → JSON 结构化技能评估报告

### 冷启动与职业指导

- **CareerBuilder + Google Cloud Jobs API**: 语义扩展 → 宽泛关键词关联具体术语
- **Agentic AI**: 感知→认知→行动→反思 四层架构的自主职业规划
- **本项目映射**: SBERT 语义冷启动 + LangGraph 四节点工作流 → 职业发展建议
